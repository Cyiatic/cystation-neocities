# CyStation Neocities

Source-controlled working copy for `https://cystation.neocities.org/`.

The first commit preserves the public crawl baseline from May 26, 2026. A complete Neocities directory export requires authenticated API access.

## AI Usage

AI assistance has been used on this project for implementation support, layout cleanup, asset preparation, deployment checks, and documentation updates. Site direction, source material, review, and final decisions remain human-led.
